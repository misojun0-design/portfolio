Google Sheets 연동 설정

한 번 설정하면 PC·모바일 어디서든 같은 데이터를 사용할 수 있습니다.

1
script.google.com 접속 → 새 프로젝트 생성
2
AppScript.gs 파일의 코드를 붙여넣고 저장 (Ctrl+S)
3
상단 메뉴 배포 → 새 배포 → 유형: 웹 앱
4
실행 계정: 나 / 액세스: 모든 사용자 → 배포
5
생성된 웹 앱 URL을 아래에 입력
URL은 이 기기에 저장됩니다. 다른 기기에서도 같은 URL을 입력하세요.
